## 概要

『 SOCCER FUN 』

サッカー専用の掲示板アプリです。掲示板・コメントを作成し情報共有ができるサービスです。

## リンク
http://www.soccerfun.site

## 制作の背景
　掲示板アプリは多くありますがサッカー専用でUIUXが整ったアプリが少なかったため今回作成してみました。
 

 初めてアプリを利用する人でも見てすぐに使えるようできるだきシンプルに作成しました。
